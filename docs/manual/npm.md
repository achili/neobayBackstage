##npm 1234


### 1.`npm install`

安装所有`package.json`中的依赖库

### 2.`npm prune`

移除所有非`package.json`中的依赖库

### 3. `nmp install xxx --save`

添加`xxx`库，并且将其写入到`package.json`中

### 4. `npm install xxx`

临时安装`xxx`库，但是不写入`package.json`中

### 5. `npm uninstall -g <package>`

删除全局模块

### 6. `npm install xxx -g`

直接就升级了
